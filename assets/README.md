# Assets

Put Vandana Sharma's production assets in this directory.

- `logo-placeholder.svg`: temporary logo used in the top bar.
- `vandana_sharma_dp.jpeg`: current profile image used in the hero.
- `profile-placeholder.svg`: old temporary profile image, safe to remove after confirming the real image.

- `resume-placeholder.txt`: temporary resume target.

When real files are ready, keep stable filenames or update `index.html` references.
